## Project

```
Insights Hub
```

## Features

- Upload PDFs
- Extract text
- Health endpoint
- Swagger documentation

## Installation

## 1. Python

Download the latest stable version from:

https://python.org

Verify:

```
python--version
```

or

```
python3--version
```

---

## 2. VS Code Extensions

Install:

- Python (Microsoft)
- Pylance
- Black Formatter
- Ruff
- Jupyter (optional)

```
pip install -r requirements.txt
```

Run

```
uvicorn app.main:app --reload
```
